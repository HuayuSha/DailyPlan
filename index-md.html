📖 使用指南

🚀 快速开始

  1. 运行 python generate-daily-plan.py
  2. 选择要创建的计划类型
  3. 在生成的 Markdown 文件中编辑您的计划
  4. 使用任何 Markdown 编辑器打开和编辑

✅ Markdown 任务语法

  • - [ ] 未完成任务
  • - [x] 已完成任务
  • - [~] 部分完成任务
  • - [-] 取消的任务
